# -Library-Management-System

## Project Setup & Installation

git clone

## Install Dependencies
npm install
npm i nodemon --save-dev

## Set Up Environment Variables
PORT=5000
DB_HOST=127.0.0.1
DB_PORT=27017
DB_NAME=lms-db
JWT_SECRET=supersecretkey
TOKEN_EXPIRES_IN=1d

## Start the Server
npm run serve
